# Draggable

### Install

```bash
npm i @ngx-starter-kit/draggable
```

### Usage

## Running unit tests

Run `ng test draggable` to execute the unit tests.

### Publish

```bash
# build deps first
ng build utils
ng build ngx-utils
# build
ng build draggable
# replace your npm key
export NPM_TOKEN="00000000-0000-0000-0000-000000000000"
# publish
npm publish dist/libs/draggable --access public
```
