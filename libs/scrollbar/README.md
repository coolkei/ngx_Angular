# Scrollbar

### Install

```bash
npm i @ngx-starter-kit/scrollbar
```

### Usage

### Running unit tests

Run `ng test app-confirm` to execute the unit tests.

### Publish

```bash
# build
ng build scrollbar
# replace your npm key
export NPM_TOKEN="00000000-0000-0000-0000-000000000000"
# publish
npm publish dist/libs/scrollbar --access public
```
