# Clap

Clap micro-interaction

### Install

```bash
npm i @ngx-starter-kit/clap
```

### Usage

### Publish

```bash
# build
ng build clap
# replace your npm key
export NPM_TOKEN="00000000-0000-0000-0000-000000000000"
# publish
npm publish dist/libs/clap --access public
```
